# Code of Conduct
SenzMatica adheres to the [CNCF Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md). All contributors, maintainers, and community members are expected to follow it when participating in project discussions, code contributions, events, and communication channels.


## Scope

This Code of Conduct applies to all project spaces, including but not limited to:

- GitHub repositories (issues, pull requests, discussions)
- Documentation
- Community forums, chats, and mailing lists
- Project-related events and meetings
## Expected Behavior

Participants are expected to:

- Be respectful and considerate in speech and actions
- Welcome diverse perspectives and experiences
- Provide constructive feedback
- Accept responsibility and apologize when mistakes occur
- Focus on what is best for the community
- Show empathy toward other community members


## Unacceptable Behavior

Unacceptable behavior includes, but is not limited to:

- Harassment, intimidation, or discrimination
- Offensive or derogatory comments related to personal characteristics
- Personal attacks, insults, or trolling
- Sexualized language or unwelcome sexual attention
- Publishing othersâ€™ private information without permission
- Sustained disruption of discussions
- Retaliation against individuals reporting Code of Conduct violations

## Protected Characteristics

The project does not tolerate harassment or discrimination based on:

- Age
- Body size
- Disability
- Ethnicity
- Gender identity or expression
- Level of experience
- Nationality
- Personal appearance
- Race
- Religion
- Sexual orientation
- Any other characteristic protected by applicable laws

## Reporting Violations

If you experience or witness unacceptable behavior, please report it confidentially.

### How to Report
- Email: info@senzmate.com
- Or contact a project maintainer directly

Reports should include:
- A description of the incident
- Date and location
- Any relevant screenshots, logs, or links
- Names of individuals involved (if known)

All reports will be handled discreetly and respectfully.


## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

Actions may include:
- A private warning
- Temporary suspension from participation
- Permanent removal from the project community

Maintainers will take appropriate action based on the severity and impact of the violation.


## No Retaliation

Retaliation against individuals who report Code of Conduct violations or participate in investigations is strictly prohibited.
